# kristi_project

React app built with Vite.

## Local development

- Install: `npm install`
- Run: `npm run dev`
- Build: `npm run build`
- Preview build: `npm run preview`

## Deploy to Netlify

This repo includes `netlify.toml` so Netlify knows how to build and serve the app:

- Build command: `npm run build`
- Publish directory: `dist`
